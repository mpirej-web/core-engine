## Installation
------------

### Prerequisites

*   C++17 compiler (e.g., GCC 9.3+, Clang 10.0+)
*   CMake 3.10+
*   Conan 1.0+
*   Docker 19.03+

### Installation Steps

1.  Clone the repository: `git clone https://github.com/core-engine/core-engine.git`
2.  Navigate to the project directory: `cd core-engine`
3.  Create a new Conan environment: `conan create.`
4.  Build the project using CMake: `cmake -B build -S.`
5.  Build the project: `cmake --build build`
6.  Run the project: `./build/core-engine`

### Docker Installation

1.  Pull the official Core Engine Docker image: `docker pull core-engine/core-engine`
2.  Run the Docker container: `docker run -it core-engine/core-engine`
